# Lab 4

Lab 4의 목표는 Rust의 [`std::collections`]를 제대로 이해하고 있는지 확인하는 것입니다.

원소의 중복 횟수(각 원소가 몇 번 나타나는지)를 추적하는 두 가지 자료구조를 구현하게 됩니다. 이 자료구조들은 기존의 일반적인 동작 위에 중복 개념을 추가한 것입니다.

첫 번째는 [`MultiSet`]으로, 중복된 원소를 가질 수 있는 집합(set)입니다.
두 번째는 [`MultiMap`]으로, 하나의 키에 여러 개의 값을 매핑할 수 있는 맵(map)입니다.

[`MultiSet`]의 경우, 필드의 타입은 제공되지 않았습니다.
