# Lab 3

Lab 3의 목표는 Rust의 **Struct**를 제대로 이해하고 있는지 확인하는 것입니다.

이 실습에서는 조커를 제외한 **표준 52장 카드 덱**의 한 장을 나타내는 [`Card`] 타입을 모델링하고 구현하게 됩니다.

카드 한 장을 표현하기 위해 필요한 필드를 정의하여 [`Card`]를 모델링합니다.
그리고, [`Card`]를 사용하는 사람이 유용하게 쓸 수 있는 몇 가지 비교 메서드를 [`Card`]에 구현합니다.
